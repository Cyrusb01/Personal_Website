Crypto World

- November 2021

Crypto World is a website designed to educate users on the utility and performance of different cryptocurrencies. This was a project done for my programming studio class and I was responsible for completing the backend. The web app is built using Flask, and uses python news api, cryptocompare, yfinance, and a backtesting library by the name of bt. The data visualization was also done by me using javascript, specifically highcharts, highstock, and plotly.

Portfolio Tools Platform

- Summer 2021

The portfolio tools platform is a collection of research tools designed to educate and arm financial advisors regarding crypto. This was my first full-stack project and involved designing, building, and deploying the app by myself. This was done for the fintech startup Onramp Invest, and is still being used as a part of their Onramp Academy. The tools allowed for easy research regarding corellation, volatilty, and performance of cryptocurrencies, and involes eight distinct tools. My favorite page is the "Custom Strategy Dashboard", in which financial advisors can pick any stocks and cryptos, along with rebalancing periods and thresholds, and be given a full breakdown on the past perfomance and statistics of the created portfolio.

Arbitrage Matrix

- March 2021

This arbitrage matrix was a small project I completed to learn and use asynchronous websockets, while also monitoring arbitrage in the crypto markets. This is a script that asynchronously subscribes to five exchanges and compares the bitcoin prices from them all in real time. Although bitcoin arbitrage is no longer a very profitable strategy, this is a great building block for other arbitrage work.

Elon Money Bot

- March 2020

Elon Money Bot is a hackathon project in which my group scraped Elon Musk's tweets in order to take advantage of his market manipulation. After scraping the latest tweet, the bot would conduct some sentiment analysis to look for any potential stocks or cryptos mentioned, along with rating it positive or negative. When a tweet comes along with a ticker and positive sentiment the bot automatically invests using the alpaca api.
